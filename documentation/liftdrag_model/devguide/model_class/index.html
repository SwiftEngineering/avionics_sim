    Introduction

    The Aerodynamic Model (AerodynamicModel) C++ class in the avionics_sim library calculates forces on an element (lift, drag, side slip drag) given a six dimensional world pose and three dimensional world velocity.

    The mathematical model implemented by this class can be read about here.

    How To Use

    Initialize Airfoil object

    The static values used by the Aerodynamic Model to calculate lift and drag forces on an element must be stored in an instance of an Airfoil class (see avionics_sim documentation for more information on the Airfoil class). Once an instance of this class has been populated, an instance of the AerodynamicModel class can be instantiated.

    Values that should remain static are element area, element lateral area, and lookup tables.

    //Vectors for holding interpolation results for each LUT.
std::vector<double> LUT_Alpha= ...;

std::vector<double> LUT_CL= ...;

std::vector<double> LUT_CD= ...;

//Area
double area_m2 = ...;

//Area
double lateralArea_m2 = ...;

//Instantiate Airfoil object
avionics_sim::Airfoil airfoil = avionics_sim::Airfoil(
    area_m2, lateralArea_m2,
    LUTalpha, LUTCL, LUTCD);

    Initialize AerodynamicModel class with Airfoil and PhysicsEnvironment objects

    Once the Airfoil object is created, create an instance of the PhysicsEnvironment object. Pass both objects into the constructor for the AerodynamicModel to instantiate an instance of it. Afterward, obtain the basis vectors for upward and forward and set them in the AerodynamicModel instance by passing them into the setBasisVectors method.

    Forward and upward vectors are presumed to be of type ignition::math::Vector3d. The environment object will provide air density information at each time step.

    
ignition::math::Vector3d forward = ...;

ignition::math::Vector3d upward = ...;

avionics_sim::PhysicsEnvironment environment = avionics_sim::PhysicsEnvironment();

aerodynamicModel = avionics_sim::AerodynamicModel(airfoil, environment);

aerodynamicModel.setBasisVectors(forward, upward);

    Update values during simulation time step, obtain resultant force

    Per time step, update the world velocity and pose and obtain air density. If air density is a static value, set the value instead during the initialization process. If there is a motor exit velocity and control angle, pass the control angle (in radians), side slip angle, planar velocity, and lateral velocity into method updateForcesInBody_N. Else, pass world velocity and pose into the method.

    If there is a motor exit velocity and control angle, pass the control angle (in radians) and motor exit velocity as both freestream and planar velocity into method updateForcesInBody_N. Else, pass world velocity and pose into the method. The result of updateForcesInBody_N is the resultant force to be applied.

    // Get world linear velocity and pose
ignition::math::Vector3d worldForces = ...;

ignition::math::Vector3d velocityInWorld_m_per_s = ...;

ignition::math::Pose3d poseInWorld_m_rad = link_->WorldPose();

ignition::math::Vector3d aeroForceOnBody_N;
  if (motorOutSub != NULL && controlJoint) {
    double propWash_m_per_s = this->motorExitVelocity_m_per_s;
    double controlAngle_rad = -controlJoint->Position();
    aeroForceOnBody_N = aerodynamicModel.updateForcesInBody_N(propWash_m_per_s, 
propWash_m_per_s, 
    RAD2DEG(controlAngle_rad), 0.0);
  } else {
    aeroForceOnBody_N = aerodynamicModel.updateForcesInBody_N(poseInWorld_m_rad,
velocityInWorld_m_per_s);
  }

    Calculation Steps

    The aerodynamic model performs steps two through eight of the ten steps in the aerodynamic mathematical model..

    Convert pose and world linear velocity into body velocity

    This is done in the method transformToLocalVelocity by calling the CoordUtils class method project_vector_global. This method is a wrapper for the method as defined in AerodynamicState. Please see documentation on this class in the avionics_sim documentation.

    transformToLocalVelocity implementation

    World velocity and pose are passed into project_vector_global, as well as a reference to the object that will contain the body velocity values.

    
//Implementation of transformToLocalVelocity within AerodynamicState class.
ignition::math::Vector3d AerodynamicModel::transformToLocalVelocity(
    ignition::math::Pose3d poseInWorld_m_rad,
    ignition::math::Vector3d velocityInWorld_m_per_s) {

      ignition::math::Vector3d velocityInBody_m_per_s;

      avionics_sim::Coordinate_Utils::project_vector_global(
        poseInWorld_m_rad,
        velocityInWorld_m_per_s,
        &velocityInBody_m_per_s);


      return velocityInBody_m_per_s;
}

    project_vector_global

    Body velocity is computed using quaternions using the following equation from the lift drag mathematical model: $$ \begin{equation}V_{body}=(\mathbf F_w \cdot \mathbf V_{world}, \mathbf R_w \cdot \mathbf V_{world}, \mathbf U_w \cdot \mathbf V_{world}) \end{equation} $$

    Implementation of body velocity calculation

    int Coordinate_Utils::project_vector_global(ignition::math::Pose3d targetFrame, 
ignition::math::Vector3d vec, 
ignition::math::Vector3d * const res)
{
    // Unit direction vectors in target (world) frame.

    //Forward is positive on x-axis in world frame. 
    ignition::math::Vector3d forward(1.0, 0.0, 0.0);

    //Upward is positive on z-axis in world frame.
    ignition::math::Vector3d upward(0.0, 0.0, 1.0);

    // Rotate into target frame to get direction unit vectors. 
    ignition::math::Vector3d forwardI = targetFrame.Rot().RotateVector(forward); 
    ignition::math::Vector3d upwardI = targetFrame.Rot().RotateVector(upward); 

    // Cross product to get right direction vector.
    ignition::math::Vector3d rightI = upwardI.Cross(forwardI).Normalize();

    // Project source vector into target frame. 
    *res = ignition::math::Vector3d(forwardI.Dot(vec), 
    rightI.Dot(vec),
    upwardI.Dot(vec)); 

    return 0; 
}

    Calculate aerodynamic parameters from the body velocity

    The aerodynamic parameters $$\alpha, \beta, and \ V_{planar}$$ are calculated from the body velocity by calling the methods calculateWindAngles, transformBodyToWindPlanar, and transformBodyToWindLateral. transformBodyToWindPlanar and transformBodyToWindLateral must be called before calculateWindAngles.

    transformBodyToWindPlanar logic implementation

    This method calculates $$V_{planar}$$ as

    $$ V_{planar} = \sqrt{{{V_{body}}_x}^2 + {{V_{body}}_z}^2} $$ 

    double AerodynamicModel::transformBodyToWindPlanar(ignition::math::Vector3d velocityInBody_m_per_s) {
  // Remove spanwise and vertical component
  ignition::math::Vector3d planarVelocity_m_per_s = ignition::math::Vector3d(
                                          velocityInBody_m_per_s.X(),
                                          0,
                                          velocityInBody_m_per_s.Z());

  // Set planar velocity.
  return planarVelocity_m_per_s.Length();
}

    transformBodyToWindLateral logic implementation

    This method calculates $$V_{lat}$$ as

    $$ V_{lat} = -{{V_{body}}_y} $$ 

    double AerodynamicModel::transformBodyToWindLateral(ignition::math::Vector3d velocityInBody_m_per_s) {
  return -velocityInBody_m_per_s.Y();
}

    calculateWindAngles logic implementation

    This method calculates the angles $$\alpha \ and \ \beta$$

    as

    $$ \begin{equation}\alpha = \frac{-u}{w}\end{equation} $$

    and

    $$ \begin{equation}\beta = arcsin(\frac{v}{V_{body}})\end{equation} $$

    where

    $$ u={{V_{body}}_x}\cdot\vec{u}_{fwd} \\ v={{V_{body}}_y}\cdot\vec{u}_{lat} \\ w={{V_{body}}_z}\cdot\vec{u}_{up} $$

    and

    $$ \vec{u}_{lat} = {\vec{u}_{up}}\times{\vec{u}_{fwd}} $$

    Implementation of wind angle calculation

    std::pair<double,double> AerodynamicModel::calculateWindAngles
(ignition::math::Vector3d velocityInBody_m_per_s) {
  //Calculate velocity in each direction.
  double u = vecUpwd.Dot(velocityInBody_m_per_s);
  double v = vecPort.Dot(velocityInBody_m_per_s);
  double w = vecFwd.Dot(velocityInBody_m_per_s);

  // Calculate alpha
  double angleOfAttack_rad = atan2(-u, w);
  double sideSlipAngle_rad = asin(v / velocityInBody_m_per_s.Length());

  return {RAD2DEG(angleOfAttack_rad), RAD2DEG(sideSlipAngle_rad)};
}

    Calculate dynamic pressure

    Dynamic pressure is calculated as a function of planar velocity in the method calculateDynamicPressure_Pa:

    $$ q_{planar} = \frac{1}{2}\rho V_{planar}^2 $$

    Implementation of dynamic pressure calculation

    double AerodynamicModel::calculateDynamicPressure_Pa(double velocity_m_per_s) {
  return 0.5 * _environment.getAirDensity_kg_per_m3() * velocity_m_per_s * velocity_m_per_s;
}

    Utilize lookup table and linear interpolation to extract coefficients of lift and drag as a function of angle of attack (alpha)

    Once the lookup tables have been set and alpha has been calculated, the Airfoil class methods calculateLiftCoefficient and calculateDragCoefficient are used respectively to obtain the coefficients of lift and drag from the lookup tables. Each function uses linear interpolation to obtain the coefficient value.

    The values in the alpha lookup table are presumed to be in degrees.

    calculateLiftCoefficient logic implementation

    Given the alpha and CL LUTs and an angle, retrieve the lift coefficient and store it in variable liftCoeff within the instance of the AerodynamicState class.

    Implementation of coefficient of lift interpolation

    double Airfoil::calculateLiftCoefficient(double angleOfAttack_deg) {
        return _aeroLUT_deg.lookup(angleOfAttack_deg, _ANGLE_OF_ATTACK_ID, _CL_ID);
}

    calculateDragCoefficient logic implementation

    Given the alpha and CD LUTs and an angle, retrieve the lift coefficient and store it in variable dragCoeff within the instance of the AerodynamicState class.

    Implementation of coefficient of drag interpolation

    double Airfoil::calculateDragCoefficient(double angleOfAttack_deg) {
        return _aeroLUT_deg.lookup(angleOfAttack_deg, _ANGLE_OF_ATTACK_ID, _CD_ID);
}

    Calculate lift and drag force scalar quantities

    This is done by calling calculateLift_N and calculateDrag_N respectively.

    calculateLift_N

    The lift scalar value is calculated according to the following equation in the mathematical model.

    $$ \begin{equation}L = C_LqA\end{equation} $$

    Implementation of coefficient of scalar lift calculation

    double AerodynamicModel::calculateLift_N(double liftCoefficient, double dynamicPressure_Pa) {
  return liftCoefficient * dynamicPressure_Pa * _airfoil.getArea_m2();
}

    calculateDrag_N

    The drag scalar value is calculated similarly:

    $$ \begin{equation}D = C_DqA\end{equation} $$

    Implementation of coefficient of scalar drag calculation

    double AerodynamicModel::calculateDrag_N(double dragCoefficient, double dynamicPressure_Pa) {
  return dragCoefficient * dynamicPressure_Pa * _airfoil.getArea_m2();
}

    Calculate drag due to side slip

    Calculation of side slip drag is performed through a call to the calculateLateralForce_N, which calculates slide slip drag as

    $$ D_{ss} = C_{D_{ss}}q_{ss}A_{lat} $$

    where $$C_{D_{ss}}$$ is coefficient of side slip drag, $$A_{lat}$$ is the lateral area of the element, and lateral dynamic pressure is $$q_{ss} = \frac{1}{2} \rho {{V_{body}}_y}^2$$.

    Implementation of coefficient of side slip drag calculation

    double AerodynamicModel::calculateLateralForce_N(double lateralDragCoefficient, 
double dynamicPressureLateral_Pa) {
  return dynamicPressureLateral_Pa * lateralDragCoefficient * _airfoil.getLateralArea_m2();
}

    Convert scalar force lift, drag, and sideslip drag representations from wind frame into body-fixed coordinate system

    To achieve this, the forces must be rotated into a body-fixed coordinate system. Rotation occurs in rotateForcesToBody. The resultant force is calculated at the end of the method.

    rotateForcesToBody

    Rotated lift and drag are calculated as follows:

    $$ F_{up} = L*cos(\alpha) + D*sin(\alpha) $$

    $$ F_{fwd} = L*sin(\alpha) - D*cos(\alpha) $$

    Implementation of lift and drag rotation

    ignition::math::Vector3d AerodynamicModel::rotateForcesToBody(double lift_N, 
double drag_N, 
double lateralForce_N, 
double angleOfAttack_deg) {
  double angleOfAttacks_rad = DEG2RAD(angleOfAttack_deg);
  double rotated_lift_N = (lift_N * cos(angleOfAttacks_rad)) + (drag_N * sin(angleOfAttacks_rad));
  double rotated_drag_N = (lift_N * sin(angleOfAttacks_rad)) - (drag_N * cos(angleOfAttacks_rad));

  //Add the forces together.
  ignition::math::Vector3d force_N = rotated_lift_N * vecUpwd + 
  rotated_drag_N * vecFwd + lateralForce_N * vecPort;

  return force_N;
}
